# 📐  Projeto API Calculadora

### API feita com Node.js para realizar operações básicas com 2 valores
- Soma ➕
- Subtração ➖
- Multiplicação ✖️
- Divisão ➗

## 🚀 Como rodar

### Clonar Repositorio:
```
git clone https://github.com/seu-usuario/api-calculadora.git
```

### Entrar na Pasta Salva a API:
```
cd api-calculadora
````

### Instalar Dependências:
```
npm install
```

### Rodar API:
```
npm start
```
